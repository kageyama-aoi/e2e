Allure レポート生成エラーの整理と正しい使い方

1. 起きていたこと（今回のエラー）

実行したコマンド：

npx allure generate allure-results --clean -o allure-report

表示されたエラー：

Unknown Syntax Error: Command not found
While running generate allure-results --clean -o allure-report
ポイント：
Allure は起動できているが、
コマンドの引数の解釈に失敗している 状態。

2. なぜこのエラーが起きたのか

原因の本質：
  • npx 経由で Allure CLI を実行している
  • PowerShell 環境
  • Allure CLI が独自のコマンドパーサを使っている
この組み合わせにより、
引数全体が「1つの不正なコマンド」として解釈された

つまり、Allure が壊れているのでも、
テスト結果がおかしいのでもありません。

3. 正しい解決方法（おすすめ）

方法①：Allure をグローバルにインストール

npm install -g allure-commandline

その後は、npx を使わずに実行します。

allure generate allure-results --clean -o allure-report
なぜこれが一番良いか
  • PowerShell / Git Bash / CI で挙動が安定
  • 引数トラブルが起きにくい
  • 実務・CI構成と同じ形になる

参考：npx を使う場合（非推奨）

npx allure -- generate allure-results -o allure-report --clean

応急処置としては動くことがありますが、
再発しやすいため常用はおすすめされません。

4. allure-results の確認

ls allure-results

中に .json ファイルがあれば OK。
なければ、テスト実行時に --alluredir が指定されていません。

5. 用語ミニ解説

CLI（Command Line Interface）

コマンド（文字）で操作するツールのこと。
allure、pytest、git などが CLI。

NPM

Node.js のパッケージ管理ツール。
ライブラリや CLI ツールを インストールする役割。 

npm install -g allure-commandline

NPX

npm に付属する 一時実行ツール。
インストールせずに、その場でコマンドを実行できる。

便利だが、CLI ツールによっては
引数解釈の相性問題が起きる。

Allure

テスト結果を人が理解しやすい HTML レポートに変換するツール。
テストは実行せず、結果を可視化する役割に専念している。

6. まとめ

  • 今回のエラーは npx と Allure CLI の相性問題
  • Allure 自体は正常に動いている
  • 実務では「グローバルインストール → allure コマンド」が安定

この理解があると、
CI（GitHub Actions / Jenkins）への展開もスムーズになります。
